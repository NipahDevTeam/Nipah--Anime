# Nipah! Anime v2.3.0

## English

This update focuses on performance, sync, UI polish, and broader platform support.

### What's new
- AniList is now the main sync provider for anime and manga
- Improved automatic progress sync and local-first retry queue
- Faster app behavior with backend cache and frontend hot-path improvements
- Better Home experience with a cleaner rotating featured hero
- BlurHash support for local covers
- Anime4K MPV preset support
- Desktop notifications
- More anime and manga source fixes and additions

### Linux support
- `.AppImage` build bundled with `WebKit2GTK 4.1`
- `.deb` package for Ubuntu/Debian-based distros
- `PKGBUILD` for Arch-based distros

### Packaging
- Updated Windows installer
- Version bumped to `2.3.0`

## Español

Esta actualización se enfoca en rendimiento, sincronización, mejoras visuales y mejor soporte multiplataforma.

### Novedades
- AniList ahora es el proveedor principal de sincronización para anime y manga
- Mejor sincronización automática de progreso y cola local con reintentos
- Mejor rendimiento general con caché en backend y optimizaciones en rutas calientes del frontend
- Mejor experiencia en Inicio con un hero destacado más limpio y rotatorio
- Soporte para BlurHash en portadas locales
- Soporte para presets Anime4K en MPV
- Notificaciones de escritorio
- Más arreglos y nuevas fuentes de anime y manga

### Soporte para Linux
- Build `.AppImage` con `WebKit2GTK 4.1` incluido
- Paquete `.deb` para Ubuntu/Debian y derivados
- `PKGBUILD` para Arch y derivados

### Empaquetado
- Instalador de Windows actualizado
- Versión actualizada a `2.3.0`
