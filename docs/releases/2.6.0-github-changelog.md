# Nipah! Anime 2.6.0

## Fixed
- Fixed Manga Online next-chapter transitions so opening the next chapter starts from the top instead of jumping to the end.
- Fixed several Manga source-selection regressions so direct-source browsing respects the chosen provider more reliably.
- Fixed MangaFire Spanish availability handling by separating `MangaFire (EN)` and `MangaFire (ES)` and resolving Spanish/Spanish (LATAM) chapter paths more safely.
- Fixed multiple provider-side manga/anime regressions that were causing inconsistent detail loading and search behavior.

## Improved
- Manga Online now has a cleaner chapter layout, unread filtering, and AniList character details.
- Anime Online now has a cleaner episode layout, cast details, and a safer MPV-first default playback flow.
- MangaFire search stays a bit lighter upstream while keeping the source available in both English and Spanish catalogs.

## Release Notes
- `MangaFire (ES)` prefers the larger Spanish branch when a title exposes both standard Spanish and Spanish (LATAM) chapter variants.
- Linux packaging metadata now matches the app version across `.deb`, AppImage, and `PKGBUILD`.

- Linux `.deb` rebuilt from the current repo state.
