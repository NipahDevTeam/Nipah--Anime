# Nipah! Anime 2.5.2

## Fixed
- Fixed Linux packaging artifacts and Arch release packaging so future releases keep safe line endings and a valid `PKGBUILD`.
- Fixed Local Anime refresh after completed downloads so imported episodes appear faster in the library.
- Fixed completed download imports to prefer AniList metadata and cover art when available.
- Fixed AnimePahe playback and downloads so it works as a stable English alternative again.
- Fixed AnimeGG landing pages, episode loading, and general playback stability across more titles.

## Improved
- Added an AnimeGG-only `Subtitles` / `Dubbed` landing-page switch for titles that expose separate variants.
- Improved AnimeGG stream selection without forcing unstable HD paths that can break playback.
- Added a dedicated anime import folder in Settings so users can drop episodes into a managed folder and scan them into Local Anime.
- Kept Downloads as history-only while imported episodes continue to live under Local Anime.

## Hidden For Stability
- Hid AnimeKai from the frontend until its playback path is stable enough to ship confidently.
- Hid the unfinished light/dark theme toggle from the public UI for this release.
