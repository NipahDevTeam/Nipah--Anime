# CachyOS Smoke Test For 2.6.0

## VM Prep
Run on a fresh CachyOS VM:

```bash
sudo pacman -Syu
sudo pacman -S --needed base-devel git go npm mpv gtk3 webkit2gtk-4.1 unzip chromium
```

## Public PKGBUILD Test
From a clean directory, using the public `2.6.0` release `PKGBUILD`:

```bash
mkdir -p ~/public-pkgtest-2.6.0
cd ~/public-pkgtest-2.6.0
makepkg -si --noconfirm -p PKGBUILD
```

## App Launch
Launch normally:

```bash
nipah-anime
```

If the VM GPU stack is weak, launch with software rendering:

```bash
LIBGL_ALWAYS_SOFTWARE=1 GSK_RENDERER=cairo WEBKIT_DISABLE_COMPOSITING_MODE=1 nipah-anime
```

## App Smoke Checklist
Confirm:

1. Home opens.
2. Search opens.
3. Manga Online opens.
4. Reported version is `2.6.0`.
5. `MPV` is detected.
6. `MangaFire (EN)` opens a title and lists chapters.
7. `MangaFire (ES)` opens a title and lists chapters.
8. Anime Online still launches MPV playback normally.

## Notes
- Public `PKGBUILD` expects the GitHub tag `v2.6.0` to exist before `makepkg` is run.
- On Linux, browser-backed provider paths may still expect a Chromium-based browser to be available.
