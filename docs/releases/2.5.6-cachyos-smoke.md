# CachyOS Smoke Test For 2.5.6

## VM Prep
Run on a fresh CachyOS VM:

```bash
sudo pacman -Syu
sudo pacman -S --needed base-devel git go npm mpv gtk3 webkit2gtk-4.1 unzip chromium
```

## Public PKGBUILD Test
From a clean directory, using the public `2.5.6` release `PKGBUILD`:

```bash
mkdir -p ~/public-pkgtest-2.5.6
cd ~/public-pkgtest-2.5.6
makepkg -si --noconfirm -p PKGBUILD
```

## App Launch
Launch normally:

```bash
nipah-anime
```

If the VM GPU stack is weak, launch with software rendering:

```bash
LIBGL_ALWAYS_SOFTWARE=1 GSK_RENDERER=cairo WEBKIT_DISABLE_COMPOSITING_MODE=1 nipah-anime
```

## App Smoke Checklist
Confirm:

1. Home opens.
2. Search opens.
3. Settings opens without a black screen.
4. Reported version is `2.5.6`.
5. `MPV` is detected.
6. AnimePahe landing pages load.
7. One AnimePahe episode opens in MPV.
8. Local Anime still opens normally.

## Notes
- On Linux, AnimePahe currently relies on a Chromium-based browser for its browser-backed DDoS-Guard flow.
- Some VM graphics stacks may need the software-rendering launch shown above even when the package itself is correct.
