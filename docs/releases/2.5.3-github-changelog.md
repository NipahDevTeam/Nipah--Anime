# Nipah! Anime 2.5.3

## Fixed
- Improved AnimeGG stream preparation so episode playback relies on the provider's real embed variants more consistently.
- Fixed several AnimeGG regressions around sub/dub stream selection and session-backed playback handoff.
- Fixed AnimeGG quality selection so the app prefers the provider's real quality ladder instead of settling too early on weaker captures.
- Added AnimeGG backup-stream handling for titles whose primary host is flaky or certificate-broken upstream.
- Kept AnimePahe stable as the main English alternative for streaming and downloads.
- Kept Arch/Linux packaging aligned with the tracked icon and LF-safe `PKGBUILD` output.

## Improved
- Improved AnimeGG playback startup and stream quality across more titles.
- Improved AnimeGG title resolution for long-running and sequel-heavy shows.
- Preserved faster Local Anime refresh after completed downloads.
- Preserved AniList-first cover art for imported/downloaded local entries where available.

## Hidden For Stability
- AnimeKai remains hidden from the public source picker until its playback path is stable enough to ship.
- The unfinished public theme toggle remains hidden from the release UI.

## Release Validation
- Windows installer rebuilt from the current repo state.
- Linux `.deb` and AppImage rebuilt from the current repo state.
- `build/bin/PKGBUILD` regenerated as LF-only for `2.5.3`.
- Arch `PKGBUILD` kept in sync with the shipped version and should be validated with `scripts/verify-arch-pkgbuild-local.sh` before publishing.
