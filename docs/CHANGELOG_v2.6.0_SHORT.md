# Nipah! Anime v2.6.0

## English

This update focuses on manga/anime online quality-of-life improvements, source handling, and release stability.

### What's new
- Fixed Manga Online next-chapter behavior so chapter transitions start at the top and progress sync behaves more reliably
- Cleaner Manga Online chapter layout with unread filtering and AniList character details
- Cleaner Anime Online episode layout with cast details and restored MPV-first playback flow
- Manga source switching is now more consistent during direct/source-driven browsing
- MangaFire now ships as separate `MangaFire (EN)` and `MangaFire (ES)` sources

### Source and stability improvements
- Improved MangaFire handling for Spanish and Spanish (LATAM) chapter availability
- Hardened provider-side search and fallback behavior for several online manga/anime paths
- More reliable Linux/Windows release packaging metadata for desktop builds

### Packaging
- Version bumped to `2.6.0`
- Updated Windows installer
- Updated Linux `.deb`, AppImage, and `PKGBUILD` release metadata

## Espanol

Esta actualizacion se enfoca en mejoras de calidad de vida para Anime Online y Manga Online, manejo de fuentes y estabilidad de lanzamiento.

### Novedades
- Se corrigio el cambio al siguiente capitulo en Manga Online para que abra desde arriba y sincronice mejor el progreso
- Interfaz de capitulos en Manga Online mas limpia, con filtro de no leidos y detalles de personajes desde AniList
- Interfaz de episodios en Anime Online mas clara, con detalles del cast y reproduccion priorizando MPV otra vez
- El cambio de fuente en Manga Online ahora es mas consistente durante la navegacion directa por proveedor
- MangaFire ahora aparece como `MangaFire (EN)` y `MangaFire (ES)`

### Mejoras de fuentes y estabilidad
- Mejor manejo de disponibilidad de capitulos en espanol y espanol LATAM para MangaFire
- Busqueda y rutas de respaldo mas solidas en varias fuentes online de manga y anime
- Metadatos de empaquetado Linux/Windows mas consistentes para los builds de escritorio

### Empaquetado
- Version actualizada a `2.6.0`
- Instalador de Windows actualizado
- Metadatos de `.deb`, AppImage y `PKGBUILD` actualizados
